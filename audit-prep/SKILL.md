# /audit-prep — 감사 대비 스킬

> 역할: 감사대비 담당
> 영역: 회계/재정

## 개요

결산자료를 준비하고, 감사 체크리스트를 점검합니다.
증빙 누락을 확인하고, 보조금 정산보고서를 작성합니다.

## 워크플로

### Step 1: 감사 유형 확인

| 유형 | 주관 | 주기 |
|------|------|------|
| 내부 감사 | 법인 이사회 | 연 1회 |
| 외부 감사 | 회계법인 | 연 1회 (일정 규모 이상) |
| 지자체 지도점검 | 시/구청 | 연 1회 |
| 보조금 정산 감사 | 보조금 교부기관 | 분기/연 |
| 시설 평가 | 사회복지협의회 | 3년 주기 |

### Step 2: 감사 체크리스트

**재정 분야:**
- [ ] 세입세출 결산서
- [ ] 예산서 및 추경 예산서
- [ ] 월별 수입/지출 원장
- [ ] 통장 사본 (전 계좌)
- [ ] 증빙 서류 편철 (월별)
- [ ] 법인카드 사용내역
- [ ] 4대보험 납부 확인서
- [ ] 원천징수 이행상황 신고서
- [ ] 보조금 교부 결정문
- [ ] 후원금 수입/사용 명세서
- [ ] 후원금 영수증 발급 대장

**인사/노무:**
- [ ] 직원 명부
- [ ] 근로계약서
- [ ] 급여대장 (원천징수 포함)
- [ ] 출퇴근 기록
- [ ] 연차 사용 현황
- [ ] 각종 수당 지급 근거

**운영:**
- [ ] 운영위원회 회의록
- [ ] 이사회 회의록
- [ ] 사업 실적 보고서
- [ ] 이용자 명부
- [ ] 프로그램 출석부

### Step 3: 증빙 누락 점검

`.swstack/budgets/expenses/`에서 지출 내역을 스캔하여:
- 증빙이 누락된 항목을 식별합니다
- 증빙 유형이 부적절한 항목을 플래그합니다
- 보완이 필요한 항목 목록을 생성합니다

### Step 4: 보조금 정산보고서

```markdown
# 보조금 정산보고서

## 1. 사업 개요
| 항목 | 내용 |
|------|------|
| 사업명 | |
| 교부기관 | |
| 교부액 | |
| 사업기간 | |

## 2. 수입 내역
| 구분 | 교부액 | 이자수입 | 기타 | 계 |
|------|--------|---------|------|---|

## 3. 지출 내역
| 항목 | 교부 예산 | 집행액 | 잔액 | 비고 |
|------|----------|--------|------|------|

## 4. 잔액 처리
- 반납액: {금액}
- 이월액: {금액}
- 사유: {사유}

## 5. 성과
{사업 성과 요약}
```

## 산출물

`.swstack/audits/` 디렉토리에 저장합니다.

## 연결 스킬

- `/budget` — 예산서 참조
- `/expense` — 지출 내역/증빙 참조
- `/report-gov` — 실적 보고 연계
