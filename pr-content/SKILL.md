# /pr-content — 홍보 콘텐츠 제작 스킬

> 역할: 홍보담당
> 영역: 홍보/대외협력

## 개요

기관 홍보 콘텐츠를 기획하고 작성합니다.
SNS 게시글, 카드뉴스 기획, 기관소식지, 보도자료를 제작합니다.

## 워크플로

### Step 1: 콘텐츠 유형 선택

| 유형 | 용도 | 채널 |
|------|------|------|
| SNS 게시글 | 일상 소식, 프로그램 안내 | 인스타그램, 블로그, 페이스북 |
| 카드뉴스 | 정보 제공, 사업 안내 | 인스타그램, 카카오톡 |
| 기관소식지 | 분기별 종합 소식 | 우편, 이메일, 홈페이지 |
| 보도자료 | 행사, 성과 언론 홍보 | 지역 언론사 |
| 모집 공고 | 프로그램 참여자/자원봉사자 모집 | 전 채널 |

### Step 2: 콘텐츠 기획

사용자에게 질문합니다:

1. 어떤 내용을 홍보하나요? (사업/행사/성과/일상)
2. 대상 독자는 누구인가요? (후원자/지역주민/참여자/언론)
3. 핵심 메시지는 무엇인가요?
4. 활용 가능한 사진/자료가 있나요?

### Step 3: 콘텐츠 작성

**SNS 게시글 구조:**
```
[훅 — 관심을 끄는 첫 문장]

[본문 — 핵심 내용 3-5줄]

[CTA — 참여/문의 안내]

#해시태그 #목록
```

**보도자료 구조:**
```
[제목 — 핵심 내용 요약]
[부제 — 보충 설명]

[리드문 — 육하원칙 요약 (1문단)]
[본문 — 상세 내용 (2-3문단)]
[인용 — 기관장/담당자 코멘트]
[기관 소개 — 보일러플레이트]
[문의처]
```

### Step 4: 개인정보 보호 확인

홍보 콘텐츠에서 반드시 확인:
- 클라이언트 사진/사례 사용 시 **서면 동의** 필수
- 아동 사진은 **보호자 동의** 필수
- 사례 소개 시 **비식별화** 처리
- "불우이웃", "어려운 이웃" 등 **낙인 표현 금지**

⚠️ 사진 속 인물의 촬영/게시 동의가 확인되지 않으면 콘텐츠를 완성하지 않습니다.
