# /pr-report — 대외 보고서 스킬

> 역할: 대외협력 담당
> 영역: 홍보/대외협력

## 개요

후원자 감사서한, 활동보고서, 후원/자원봉사 현황 리포트를 작성합니다.

## 워크플로

### Step 1: 보고서 유형 선택

| 유형 | 대상 | 주기 |
|------|------|------|
| 후원자 감사서한 | 개인/단체 후원자 | 후원 즉시/분기 |
| 후원금 사용 보고서 | 후원자 | 분기/연간 |
| 활동보고서 | 후원자/이사회/지자체 | 월간/분기/연간 |
| 자원봉사 현황 리포트 | 자원봉사센터/기관 내부 | 월간 |
| 기관 연간보고서 | 전체 이해관계자 | 연간 |

### Step 2: 내용 구성

**후원금 사용 보고서:**
```markdown
# 후원금 사용 보고서 ({기간})

## 후원 현황
- 총 후원 금액: {금액}원
- 후원자 수: {명/단체}
- 후원 유형: 정기후원 {금액} / 일시후원 {금액}

## 사용 내역
| 사업명 | 사용 금액 | 수혜 인원 | 주요 성과 |
|--------|----------|----------|----------|

## 대표 활동 사례
{비식별화된 사례 2-3건 — 변화 스토리}

## 감사 인사
{진심어린 감사 메시지}
```

**활동보고서:**
- 사업별 실적 (목표 대비 달성률)
- 주요 활동 사진 및 설명
- 참여자 만족도
- 향후 계획

### Step 3: 개인정보 보호 확인

- 클라이언트 사례 인용 시 반드시 비식별화
- 사진 게시 동의 확인
- 후원자 명단 공개 시 동의 확인

## 산출물

`.swstack/pr/reports/` 디렉토리에 저장합니다.

## 연결 스킬

- `/budget` — 후원금 사용 내역 참조
- `/expense` — 집행 내역 참조
- `/pr-content` — 보고서 기반 홍보물 제작
