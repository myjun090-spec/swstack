# /law-check — 법령/윤리 검토 스킬

> 역할: 법률/윤리 자문
> 유형: 유틸리티

## 개요

사회복지 실천 관련 법령과 윤리강령을 검토합니다.
의무신고, 개인정보보호, 서비스 제공 기준 등을 확인합니다.

## 주요 참조 법령

### 기본법
| 법령 | 핵심 내용 |
|------|----------|
| 사회복지사업법 | 시설 운영, 사회복지사 자격, 비밀누설 금지 |
| 사회보장급여법 | 사회보장급여 신청/조사/결정/제공 |

### 대상별 법률
| 법령 | 핵심 내용 |
|------|----------|
| 아동복지법 | 아동학대 신고의무, 보호조치 |
| 노인복지법 | 노인학대 신고의무, 노인복지시설 |
| 장애인복지법 | 장애인학대 신고의무, 활동지원 |
| 한부모가족지원법 | 한부모가족 지원 기준 |
| 다문화가족지원법 | 다문화가족 서비스 |
| 정신건강복지법 | 입원/퇴원 절차, 권리 |
| 노숙인복지법 | 노숙인 지원 체계 |

### 공통 법률
| 법령 | 핵심 내용 |
|------|----------|
| 개인정보보호법 | 수집/이용/제공/파기 |
| 긴급복지지원법 | 긴급 지원 요건/절차 |
| 자살예방법 | 자살위험자 발견 시 조치 |
| 가정폭력방지법 | 신고의무, 피해자 보호 |

### 윤리강령
- 한국사회복지사 윤리강령
- 사회복지사 선서

## 워크플로

### 주요 기능

1. **의무신고 대상 확인**: 상황을 입력하면 신고의무 해당 여부, 신고처, 절차 안내
2. **서비스 제공 기준 확인**: 특정 서비스의 법적 근거, 대상 기준 확인
3. **윤리적 딜레마 분석**: 관련 윤리강령 조항 및 판단 가이드 제시
4. **시설 운영 기준 확인**: 인력 기준, 시설 기준, 운영 기준 확인

### 의무신고 판단 가이드

```
⚠️ 의무신고 판단

상황: {입력된 상황}

1. 신고의무 해당 여부: 예/아니오
2. 관련 법률: {법률명 제O조}
3. 신고처: {기관명 및 연락처}
4. 신고 기한: {기한}
5. 신고 방법: {방법}
6. 주의사항: {법적 유의사항}

※ 이 판단은 참고용이며, 확실하지 않은 경우
   법률 전문가 또는 관할 기관에 문의하시기 바랍니다.
```

## 연결 스킬

- `/privacy-guard` — 개인정보보호 관련 법령
- `/intake` — 접수 시 의무신고 대상 확인
- `/assessment` — 사정 시 법적 조치 필요 여부
