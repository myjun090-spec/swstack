# /law-check -- 법령/윤리 검토 스킬 (국가법령정보 MCP)

> 역할: 법률/윤리 자문
> 유형: 유틸리티

## 개요

국가법령정보 MCP 도구를 활용하여 사회복지 실천 관련 법령, 판례, 유권해석을 실시간으로 검토합니다.
의무신고, 개인정보보호, 서비스 제공 기준, 시설 운영 기준 등을 현행 법령 기준으로 확인합니다.

## 사용 가능한 MCP 도구

| 도구 | 용도 | 사용 시점 |
|------|------|----------|
| `search_law` | 키워드로 관련 법령 검색 | 어떤 법령이 관련되는지 모를 때 |
| `get_three_tier` | 법률-시행령-시행규칙 3단 비교 | 구체적 기준(인력, 시설, 급여액)을 확인할 때 |
| `chain_full_research` | 법령+판례+해석례 통합 검색 | 복잡한 법적 쟁점을 종합 검토할 때 |
| `get_annexes` | 별표/서식 조회 | 시설기준, 인력기준, 급여액표, 서식을 확인할 때 |
| `search_precedent` | 관련 판례 검색 | 법 해석이 불분명하거나 분쟁 소지가 있을 때 |
| `search_ordinance` | 지자체 조례 검색 | 지역별 차이가 있는 기준을 확인할 때 |
| `get_legal_terms` | 법률 용어 설명 | 법률 용어의 정확한 정의를 확인할 때 |
| `search_interpretation` | 유권해석 검색 | 법령 적용에 대한 정부 공식 해석을 확인할 때 |

## 워크플로

### Step 1: 질의 분석

사용자의 질문을 분석하여 다음을 파악합니다:
- 어떤 법령 영역에 해당하는지 (신고의무, 시설기준, 급여, 자격 등)
- 어떤 대상자 유형인지 (아동, 노인, 장애인, 한부모 등)
- 어떤 수준의 답변이 필요한지 (조문 확인, 기준표 확인, 판례 확인 등)

### Step 2: 적절한 MCP 도구 선택 및 호출

질의 유형에 따라 도구를 선택합니다:

#### 의무신고 관련 질문
1. `search_law`로 관련 법령 검색 (예: "아동학대 신고의무")
2. `get_three_tier`로 법률-시행령-시행규칙 3단 비교 (신고 절차, 기한, 벌칙 확인)
3. 필요시 `search_precedent`로 관련 판례 확인

#### 시설 운영 기준 질문
1. `search_law`로 관련 법령 검색 (예: "사회복지시설 인력기준")
2. `get_annexes`로 별표 조회 (인력배치기준표, 시설설치기준표)
3. `search_ordinance`로 지자체별 추가 기준 확인

#### 서비스 제공 기준 질문
1. `search_law`로 관련 법령 검색
2. `get_three_tier`로 구체적 기준 확인
3. `search_interpretation`로 유권해석 확인

#### 법적 쟁점/딜레마 질문
1. `chain_full_research`로 법령+판례+해석례 통합 검색
2. 필요시 `get_legal_terms`로 핵심 용어 정의 확인

### Step 3: 결과 정리

```
[법령 검토 결과]

질의: {사용자 질문 요약}

1. 관련 법령
   - {법령명} 제{조}조 ({조문 제목})
   - 핵심 내용: {요약}

2. 구체적 기준 (해당 시)
   - 시행령/시행규칙 기준: {내용}
   - 별표/서식: {해당 별표 내용 요약}

3. 판례/해석례 (해당 시)
   - {판례/해석례 요약}

4. 지자체 조례 (해당 시)
   - {조례 기준 차이점}

5. 실무 적용 가이드
   - {구체적 행동 지침}

※ 이 검토는 참고용이며, 법적 분쟁이 예상되는 경우
  법률 전문가에게 자문을 구하시기 바랍니다.
  (검토 시점: {날짜})
```

## 주요 복지 법령 키워드 가이드

상황별로 검색할 법령 키워드입니다:

### 신고의무 관련
- 아동학대: "아동복지법 신고의무", "아동학대범죄의 처벌 등에 관한 특례법"
- 노인학대: "노인복지법 신고의무"
- 장애인학대: "장애인복지법 신고의무"
- 가정폭력: "가정폭력방지 및 피해자보호 등에 관한 법률"
- 자살위험: "자살예방 및 생명존중문화 조성을 위한 법률"

### 서비스/급여 관련
- 기초생활: "국민기초생활보장법"
- 긴급지원: "긴급복지지원법"
- 장애인활동지원: "장애인활동 지원에 관한 법률"
- 노인돌봄: "노인장기요양보험법"

### 시설 운영 관련
- 시설 설치/운영: "사회복지사업법 시설"
- 재무/회계: "사회복지시설 재무회계규칙" (별표로 서식 확인)
- 인력기준: 각 법률 시행규칙 별표 (get_annexes 활용)

### 개인정보 관련
- 수집/이용/제공: "개인정보보호법"
- 민감정보: "개인정보보호법 민감정보"

## 윤리강령

법령 검토와 함께 다음 윤리 기준도 고려합니다:
- 한국사회복지사 윤리강령
- 사회복지사 선서
- 기관 내부 윤리 규정

윤리적 딜레마 분석 시에는 `chain_full_research`를 통해 관련 법령과 판례를 종합적으로 검토한 후, 윤리강령 조항과 대조하여 판단 가이드를 제시합니다.

## 연결 스킬

- `/law-research` -- 특정 복지 서비스/대상자에 대한 법령 심층 조사
- `/privacy-guard` -- 개인정보보호 관련 법령
- `/intake` -- 접수 시 의무신고 대상 확인
- `/assessment` -- 사정 시 법적 조치 필요 여부
- `/audit-prep` -- 감사대비 시 관련 법령 기준 확인
