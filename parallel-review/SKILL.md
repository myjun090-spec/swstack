# /parallel-review — 다학제 병렬 사례검토 스킬

> 역할: 다학제 검토 패널 코디네이터
> 유형: 멀티에이전트 파이프라인 (병렬 실행)

## 개요

하나의 사례를 4명의 전문 에이전트가 동시에 병렬로 검토합니다.
각 에이전트는 고유한 전문 관점에서 독립적으로 분석하며, 최종적으로 코디네이터가 모든 검토 결과를 통합하여 종합 보고서를 작성합니다.
SalesLens의 멀티에이전트 패턴 중 "병렬 실행 + 결과 통합" 구조를 사회복지 사례검토에 적용했습니다.

**활용 시점**: 복잡 사례, 위기 사례, 종결 전 최종 검토, 슈퍼비전 보고 등

## 실행 조건

- 검토할 사례의 기록이 존재해야 합니다 (최소 intake.md)
- 더 많은 기록(assessment, plan, notes)이 있을수록 검토가 풍부해집니다

## 병렬 에이전트 구조

```
                    ┌─── [법률 검토 에이전트] ───────┐
                    │                               │
[사례 기록] ───────├─── [심리 평가 에이전트] ───────┼──→ [통합 검토 보고서]
                    │                               │
                    ├─── [자원 연계 에이전트] ───────┤
                    │                               │
                    └─── [위기 평가 에이전트] ───────┘
```

4개 에이전트가 **동일한 사례 기록을 동시에** 읽고, 각자의 전문 관점에서 독립적으로 분석합니다.

## 워크플로

### Step 1: 사례 선택 및 기록 로드

사용자에게 다음을 질문합니다:

1. **검토 대상 사례**: case-id 또는 파일 경로
2. **검토 목적**: 신규 사정 보완 / 개입 방향 검토 / 종결 전 점검 / 위기 대응 / 슈퍼비전 자료
3. **특별히 중점 검토할 영역**: (선택사항)

사례의 모든 기록을 로드합니다:
- `intake.md` (접수기록서)
- `assessment.md` (사정보고서)
- `plan.md` (개입계획서)
- `notes/` (상담기록)
- `conferences/` (사례회의록)
- `evaluation.md` (평가보고서)

---

### Step 2: 4개 에이전트 병렬 검토

#### 에이전트 1: 법률 검토 에이전트

**전문 영역**: 법적 의무, 권리 보호, 법적 위험

**검토 항목**:
1. **의무신고 해당 여부**
   - 아동학대 (아동복지법 제10조의4)
   - 노인학대 (노인복지법 제39조의6)
   - 장애인학대 (장애인복지법 제59조의4)
   - 가정폭력 (가정폭력방지법 제4조)
   - 자살위험 (자살예방법)

2. **클라이언트 권리 보호**
   - 동의서 적정성 (개인정보 수집/제공 동의)
   - 자기결정권 보장 여부
   - 알권리 보장 (서비스 내용, 한계 등 고지)

3. **기관의 법적 의무**
   - 비밀보장 의무 준수 (사회복지사업법 제47조)
   - 기록 보관 의무
   - 보조금 사용 적정성 (해당 시)

4. **법적 리스크 평가**
   - 소송 가능성이 있는 영역
   - 행정 처분 위험 영역
   - 사전 조치 권고

**산출물**: 법률 검토 의견서

---

#### 에이전트 2: 심리 평가 에이전트

**전문 영역**: 심리/정서적 상태, 정신건강, 심리적 개입 적절성

**검토 항목**:
1. **심리/정서 상태 분석**
   - 현재 정서 상태 평가 (기록에 기반)
   - 척도 점수 검토 및 해석 (PHQ-9, GAD-7 등)
   - 트라우마 이력 및 영향
   - 심리적 방어기제 패턴

2. **심리적 개입 적절성**
   - 적용된 상담 이론/기법의 적합성
   - 치료적 관계(라포) 형성 수준
   - 개입 강도/빈도의 적절성
   - 대안적 접근법 제안

3. **심리적 위험 평가**
   - 자살/자해 위험도 (Columbia Suicide Severity Rating Scale 기준)
   - 약물/알코올 남용 위험
   - 정서적 악화 징후
   - 전문 치료 의뢰 필요성

4. **심리적 강점 및 회복탄력성**
   - 대처 능력 평가
   - 동기 수준
   - 자기효능감
   - 사회적 지지의 심리적 효과

**산출물**: 심리 평가 의견서

---

#### 에이전트 3: 자원 연계 에이전트

**전문 영역**: 지역사회 자원, 서비스 연계, 복지 제도

**검토 항목**:
1. **현재 서비스 적절성**
   - 제공 중인 서비스가 욕구에 부합하는가
   - 서비스 중복이나 공백은 없는가
   - 서비스 접근성은 적절한가

2. **추가 연계 가능 자원**
   - 미연계 자원 중 활용 가능한 것
   - 중앙정부/지자체 복지 제도 활용 가능성
   - 민간 자원 (후원, 자원봉사 등)
   - 자조 모임, 지역사회 네트워크

3. **연계 효과성 분석**
   - 기존 연계 서비스의 효과
   - 연계 과정의 문제점
   - 연계기관과의 협력 수준

4. **자립 지원 방안**
   - 장기적 자립을 위한 자원 계획
   - 클라이언트 역량 강화 자원
   - 비공식 지지체계 확대 방안

**산출물**: 자원 연계 의견서

---

#### 에이전트 4: 위기 평가 에이전트

**전문 영역**: 위기 상황 평가, 안전 계획, 긴급 대응

**검토 항목**:
1. **현재 위기 수준 평가**
   - 즉각적 위험 (신체적 안전, 생명 위협)
   - 단기 위험 (48시간 이내 악화 가능성)
   - 중장기 위험 (재발, 만성화 가능성)

2. **위기 요인 분석**
   | 영역 | 위기 요인 | 보호 요인 | 현재 수준 |
   |------|----------|----------|----------|
   | 신체적 안전 | | | 안전/주의/위험/긴급 |
   | 정신건강 | | | 안전/주의/위험/긴급 |
   | 경제적 위기 | | | 안전/주의/위험/긴급 |
   | 주거 안정 | | | 안전/주의/위험/긴급 |
   | 가정 내 위험 | | | 안전/주의/위험/긴급 |

3. **안전 계획 검토**
   - 현재 안전 계획의 충분성
   - 비상연락체계의 적절성
   - 보호 요인 활용도

4. **긴급 대응 권고**
   - 즉시 조치가 필요한 사항
   - 모니터링 강화 영역
   - 예방적 개입 제안

**산출물**: 위기 평가 의견서

---

### Step 3: 결과 통합 — 코디네이터

4개 에이전트의 검토 결과를 통합합니다:

1. **공통 소견 도출**: 2개 이상 에이전트가 동일하게 지적한 사항
2. **상충 의견 정리**: 에이전트 간 의견이 다른 영역과 그 이유
3. **종합 우선순위 결정**: 모든 검토 결과를 종합한 행동 우선순위
4. **통합 권고안 작성**: 실천가를 위한 구체적 행동 지침

## 개인정보 보호 규칙

- 모든 에이전트는 `/privacy-guard` 원칙을 준수합니다
- 검토 보고서에 클라이언트 실명은 포함하지 않습니다
- 외부 공유 시 추가적인 비식별화 처리를 안내합니다

## 연결 스킬

- **데이터 소스**: 모든 사례관리 스킬의 산출물 (`/intake`, `/assessment`, `/plan-case`, `/intervene`, `/evaluate`)
- **보조**: `/law-check` — 법률 에이전트의 상세 법률 검토 시 참고
- **보조**: `/resource-map` — 자원 에이전트의 자원 검색 시 참고
- **보조**: `/knowledge-query` — 유사 사례 검색으로 검토 보완
- **후속**: `/case-conference` — 검토 결과를 사례회의에 상정
- **후속**: `/plan-case` — 검토 결과를 반영한 계획 수정
- **보호**: `/privacy-guard` — 전체 검토에서 암묵적 활성화
