# /privacy-guard — 개인정보 보호 스킬

> 역할: 개인정보보호 책임자
> 유형: 유틸리티 (모든 스킬에서 암묵적으로 활성화)

## 개요

모든 swstack 산출물에서 개인정보를 보호합니다.
gstack의 `/careful` 스킬에 해당하며, 파괴적 행동 대신 **개인정보 유출**을 방지합니다.

## 항시 활성화 규칙

이 스킬의 규칙은 `/privacy-guard`를 명시적으로 실행하지 않아도 **모든 스킬에서 자동 적용**됩니다.

### 1. 자동 마스킹 대상

다음 정보가 산출물에 포함되면 자동으로 마스킹합니다:

| 정보 유형 | 원본 예시 | 마스킹 결과 |
|----------|----------|------------|
| 실명 | 김철수 | 클라이언트 A / CT-A |
| 주민등록번호 | 850101-1234567 | ⚠️ 기록 불가 — 별도 시스템 사용 |
| 전화번호 | 010-1234-5678 | ⚠️ 기록 불가 — 별도 시스템 사용 |
| 상세주소 | 서울시 강남구 역삼동 123-4 | 서울시 강남구 (동 단위까지만) |
| 이메일 | kim@email.com | ⚠️ 기록 불가 — 별도 시스템 사용 |
| 계좌번호 | 국민 123-456-789 | ⚠️ 기록 불가 — 별도 시스템 사용 |
| 건강정보 상세 | HIV 양성 | 민감건강정보 — 별도 보안기록 |

### 2. 기록 가능한 정보 (비식별 처리)

| 정보 유형 | 기록 방식 |
|----------|----------|
| 성별 | 남/여/기타 |
| 연령 | 연령대 (예: 40대 후반) |
| 가구 구성 | 유형 (예: 1인가구, 조손가구) |
| 거주 지역 | 구/동 단위 |
| 소득 수준 | 구간 (예: 기초수급, 차상위, 중위소득 50% 이하) |
| 장애 | 유형/등급 (예: 지체장애 3급) |

### 3. 제3자 제공 체크

다른 기관에 정보를 전달하는 산출물(연계 의뢰서 등) 작성 시:

```
⚠️ 제3자 정보 제공 확인
- 제공 대상 기관: {기관명}
- 제공 정보 범위: {구체적 항목}
- 정보 제공 동의서 징구 여부: [ ] 확인 필요

동의서가 확인되지 않으면 정보 제공이 포함된 문서를 생성하지 않습니다.
계속하시려면 동의 확인 여부를 알려주세요.
```

### 5. 의무신고 예외

개인정보 보호에도 불구하고, 다음 상황에서는 **신고 의무**가 우선합니다:

- 아동학대/방임 의심
- 노인학대/방임 의심
- 장애인학대 의심
- 자살/자해 긴박한 위험
- 타인에 대한 긴박한 위해

이 경우 실천가에게 즉시 알립니다:
```
🚨 의무신고 대상 감지
감지 내용: {구체적 내용}
관련 법령: {해당 법령 및 조항}
신고처: {관할 기관 및 연락처}

의무신고는 개인정보보호법보다 우선합니다.
신고 절차를 안내할까요?
```

## 명시적 실행 시 추가 기능

`/privacy-guard`를 직접 실행하면 다음 추가 기능을 제공합니다:

### 기존 산출물 점검

지정된 case-id의 모든 파일을 스캔하여:
1. 마스킹되지 않은 개인정보를 탐지합니다
2. 발견된 항목을 목록으로 보여줍니다
3. 자동 마스킹 처리를 제안합니다

### 동의서 현황 점검

```markdown
## 동의서 현황 — 사례 {case-id}

| 동의 항목 | 징구 여부 | 징구일 | 만료일 | 상태 |
|----------|----------|--------|--------|------|
| 개인정보 수집·이용 | ✅ | 2026-01-15 | 2027-01-14 | 유효 |
| 제3자 제공 (A기관) | ✅ | 2026-01-15 | 2026-07-14 | ⚠️ 갱신필요 |
| 제3자 제공 (B기관) | ❌ | - | - | 미징구 |
| 사진/영상 촬영 | ❌ | - | - | 미징구 |
```

### 정보 파기 안내

종결 사례에 대해:
- 보존 기간 확인 (사회복지사업법: 5년)
- 파기 대상 식별
- 파기 방법 안내 (전자: 복구불가능 삭제 / 종이: 파쇄)

## 낙인 방지 언어 검사

산출물에서 다음 표현을 감지하고 대안을 제시합니다:

| ❌ 사용 금지 | ✅ 대안 표현 |
|-------------|-------------|
| 결손가정 | 한부모가정 |
| 불우이웃 | 지원이 필요한 이웃 |
| 장애우 | 장애인 |
| 다문화가정 (맥락 부적절 시) | 이주배경가정 |
| 정신병자 | 정신건강 어려움을 겪는 분 |
| 노숙자 | 주거취약계층 / 거리생활인 |
| 미혼모 | 비혼모 / 한부모 |
| 수급자 (비하적 맥락) | 기초생활수급 대상자 |
| 알코올 중독자 | 알코올 사용 장애를 가진 분 |

## 연결 스킬

- 모든 스킬에서 암묵적으로 활성화
- `/law-check` — 상세 법령 검토 필요 시
- `/close-case` — 종결 시 정보 파기 절차 연계
