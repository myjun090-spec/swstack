# /template — 기관 양식 관리 스킬

> 역할: 양식 관리 담당
> 유형: 유틸리티

## 개요

기관에서 사용하는 각종 양식을 관리하고 생성합니다.

## 사용 가능한 양식 목록

| 번호 | 양식명 | 용도 |
|------|--------|------|
| 1 | 접수기록서 | 초기 상담 접수 |
| 2 | 욕구사정지 | 클라이언트 사정 |
| 3 | 서비스계획서 | 개입계획 |
| 4 | 상담기록지 | 회기별 기록 |
| 5 | 사례회의록 | 사례회의 |
| 6 | 평가보고서 | 사례 평가 |
| 7 | 종결요약서 | 사례 종결 |
| 8 | 동의서 | 개인정보/서비스 동의 |
| 9 | 의뢰서 | 타 기관 의뢰 |
| 10 | 품의서 | 지출 품의 |
| 11 | 행사기획서 | 행사 기획 |
| 12 | 회의록 | 일반 회의 |

## 사용법

```
/template {양식번호 또는 양식명}
```

지정된 양식의 빈 템플릿을 생성합니다.
기관 정보(기관명, 주소, 대표자 등)가 설정되어 있으면 자동으로 채웁니다.

## 기관 정보 설정

최초 사용 시 기관 기본 정보를 `.swstack/config.md`에 저장합니다:

```markdown
---
org_name: {기관명}
org_type: {종합사회복지관/노인복지관/장애인복지관/지역자활센터/...}
org_address: {주소}
org_tel: {전화번호}
org_director: {관장/센터장명}
region: {관할 지역}
---
```

## 산출물

요청된 양식을 현재 디렉토리 또는 지정 경로에 생성합니다.
