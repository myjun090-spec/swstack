# /budget — 예산 편성 스킬

> 역할: 회계담당
> 영역: 회계/재정

## 개요

사업별 예산을 편성하고 관리합니다.
보조금 항목별 배분, 예산 수정, 집행률 모니터링을 수행합니다.

## 워크플로

### Step 1: 예산 구조 파악

사용자에게 다음을 질문합니다:

1. 사업명 및 사업 기간
2. 재원 유형: 정부보조금 / 지자체보조금 / 법인전입금 / 후원금 / 자부담
3. 총 예산 규모
4. 예산 편성 기준 (보조금 교부조건, 기관 규정 등)

### Step 2: 항목별 예산 편성

사회복지시설 보조금 항목 기준:

| 관 | 항 | 목 | 예산액 | 산출근거 |
|----|----|----|--------|----------|
| 인건비 | 급여 | 기본급 | | 호봉표 기준 |
| | | 제수당 | | |
| | 퇴직적립금 | | | 급여총액 × 1/12 |
| | 4대보험 | | | 기관부담분 |
| 사업비 | {사업명} | 강사비 | | 단가 × 횟수 |
| | | 재료비 | | |
| | | 여비/교통비 | | |
| | | 홍보비 | | |
| 운영비 | 사무용품 | | | |
| | 공공요금 | | | |
| | 차량유지비 | | | |

### Step 3: 교차 검증

- 보조금 교부조건과 항목 일치 여부 확인
- 항목 간 전용 제한 확인 (인건비→사업비 전용 불가 등)
- 총액 검증 (항목 합계 = 총 예산)

### Step 4: 예산 수정 (변경)

기존 예산 수정 시:
- 변경 사유 기록
- 변경 전/후 비교표 생성
- 전용 가능 여부 확인
