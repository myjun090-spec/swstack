# /minutes — 회의록 스킬

> 역할: 서기
> 영역: 행정

## 개요

각종 회의(운영위원회, 팀 회의, 직원회의 등)의 회의록을 작성합니다.
결정사항 추출, 후속 조치 할당, 기한 추적을 수행합니다.

## 워크플로

### Step 1: 회의 정보

| 항목 | 내용 |
|------|------|
| 회의명 | |
| 일시 | |
| 장소 | |
| 참석자 | |
| 불참자 | |
| 작성자 | |

### Step 2: 회의록 작성

사용자가 회의 내용을 구술하거나 입력하면 구조화합니다:

```markdown
# 회의록 — {회의명}

## 회의 정보
{기본 정보 테이블}

## 안건 및 논의 내용

### 안건 1: {제목}
- 보고/제안 내용:
- 논의 내용:
- 결정 사항:

### 안건 2: {제목}
...

## 결정 사항 요약
| 번호 | 결정 내용 | 담당 | 기한 | 상태 |
|------|----------|------|------|------|

## 기타/전달 사항

## 다음 회의 일정
```

### Step 3: 후속 조치 추적

이전 회의록의 결정사항 이행 여부를 확인합니다.

## 산출물

`.swstack/minutes/` 디렉토리에 저장합니다.
