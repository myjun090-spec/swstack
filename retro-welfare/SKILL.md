# /retro-welfare — 사업 회고 스킬

> 역할: 팀장
> 영역: 슈퍼비전

## 개요

월간/분기별 사업 회고를 수행합니다.
사업별 성과 분석, 팀 역량 개발 계획, 소진 예방을 포함합니다.
gstack의 `/retro`에 해당합니다.

## 워크플로

### Step 1: 데이터 수집

자동 집계:
- 사례관리 실적 (신규/진행/종결)
- 프로그램 실적 (참여율, 만족도)
- 예산 집행률
- 상담 회기 수

### Step 2: 회고 질문

팀원들에게 묻습니다:

1. **잘한 것 (Keep)**: 이번 기간에 계속하고 싶은 것은?
2. **문제점 (Problem)**: 어려웠거나 개선이 필요한 것은?
3. **시도할 것 (Try)**: 다음 기간에 새로 시도하고 싶은 것은?
4. **배운 것 (Learn)**: 이번 기간에 배운 것은?

### Step 3: 회고 보고서

```markdown
# {기간} 사업 회고

## 1. 실적 요약
{핵심 지표 대시보드}

## 2. KPT 분석
### Keep (유지)
### Problem (문제)
### Try (시도)

## 3. 팀 상태
- 팀원별 업무량 분석
- 소진 위험도 체크
- 역량 개발 현황

## 4. 다음 기간 우선순위
1. {최우선 과제}
2. {차순위 과제}
3. {장기 과제}

## 5. 실행 계획
| 과제 | 담당 | 기한 | 상태 |
|------|------|------|------|
```

## 산출물

`.swstack/retros/` 디렉토리에 저장합니다.
