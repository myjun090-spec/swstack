# /resource-map — 지역사회 자원 관리 스킬

> 역할: 자원관리 담당
> 유형: 유틸리티

## 개요

지역사회 자원 DB를 관리하고 검색합니다.
연계기관 정보, 서비스 내용, 연락처를 체계적으로 관리합니다.

## 워크플로

### Step 1: 자원 검색

사용자가 욕구 영역을 입력하면 관련 자원을 검색합니다:

| 욕구 영역 | 관련 자원 유형 |
|----------|---------------|
| 경제 | 긴급복지, 자활센터, 푸드뱅크, 사회적기업 |
| 주거 | 주거복지센터, 임대주택, 쉼터 |
| 의료 | 보건소, 무료진료소, 정신건강복지센터 |
| 교육 | 학습지원, 장학재단, 방과후교실 |
| 돌봄 | 재가서비스, 돌봄센터, 주간보호 |
| 법률 | 법률구조공단, 법률홈닥터, 마을변호사 |
| 심리 | 정신건강복지센터, 상담센터, 자살예방센터 |
| 취업 | 고용센터, 직업훈련, 사회적기업 |

### Step 2: 자원 등록/수정

```markdown
---
resource_id: {ID}
name: {기관명}
category: {분야}
updated: {날짜}
---

# {기관명}

- 분야: {분야}
- 주소: {주소}
- 연락처: {전화}
- 운영시간: {시간}
- 대상: {이용 대상}
- 서비스: {제공 서비스}
- 연계 방법: {의뢰서/전화/방문 등}
- 비용: {무료/유료}
- 비고: {특이사항}
```

### Step 3: 자원 현황 리포트

분야별, 지역별 자원 현황을 요약합니다.

## 산출물

`.swstack/resources/` 디렉토리에 저장합니다.

## 연결 스킬

- `/assessment` — 사정 시 연계 가능 자원 조회
- `/plan-case` — 서비스 연계 계획
- `/intervene` — 연계 기록
