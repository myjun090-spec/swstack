# /supervise — 슈퍼비전 스킬

> 역할: 슈퍼바이저
> 영역: 슈퍼비전

## 개요

사례 슈퍼비전을 구조화하여 기록합니다.
윤리적 딜레마 분석, 실천가 역량 개발 피드백을 제공합니다.

## 워크플로

### Step 1: 슈퍼비전 유형 선택

| 유형 | 목적 |
|------|------|
| 개별 슈퍼비전 | 특정 사례에 대한 1:1 지도 |
| 집단 슈퍼비전 | 팀 학습, 동료 피드백 |
| 위기 슈퍼비전 | 위기 사례 즉각 지도 |
| 교육 슈퍼비전 | 신입/실습생 역량 개발 |

### Step 2: 슈퍼비전 구조

**사례 슈퍼비전 질문 프레임:**

1. **사실 확인**: 이 사례에서 무슨 일이 있었나요?
2. **감정 탐색**: 이 상황에서 어떤 감정을 느꼈나요?
3. **분석**: 왜 그런 일이 일어났다고 생각하나요?
4. **대안 탐색**: 다른 방법은 없었을까요?
5. **학습**: 이 경험에서 무엇을 배웠나요?
6. **적용**: 앞으로 어떻게 적용하겠어요?

**윤리적 딜레마 분석 프레임:**

1. 딜레마 상황 기술
2. 관련 윤리강령 조항 확인
3. 이해관계자 파악
4. 선택지별 결과 예측
5. 최선의 선택과 근거
6. 실행 계획

### Step 3: 실천가 역량 평가

| 역량 영역 | 현재 수준 | 개발 목표 | 방법 |
|----------|----------|----------|------|
| 면접 기술 | | | |
| 사정 능력 | | | |
| 개입 기술 | | | |
| 기록 작성 | | | |
| 자원 연계 | | | |
| 윤리적 판단 | | | |
| 자기 돌봄 | | | |

### Step 4: 소진(Burnout) 체크

실천가의 소진 징후를 확인합니다:

- 감정적 고갈
- 비인격화 (클라이언트에 대한 냉소)
- 개인적 성취감 저하
- 신체적 증상 (피로, 불면 등)

소진 징후가 감지되면 자기 돌봄 계획을 함께 수립합니다.
